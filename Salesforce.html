Course Details

Salesforce is a cloud-based customer relationship management (CRM) platform that helps businesses manage and analyse customer interactions and data throughout the customer lifecycle. It provides a suite of applications, such as Sales Cloud, Service Cloud, and Marketing Cloud, that is designed to help businesses manage their sales, customer service, and marketing activities.

Course Details

More details

  • 01. Basics of the Salesforce platform

    Understand the basics of the Salesforce platform and its components, such as Sales Cloud, Service Cloud, and Marketing Cloud.

  • 02. Learn about the Salesforce data models

    Learn about the Salesforce data model, including objects, fields, relationships, and standard and custom objects.

  • 03. Learn about the Salesforce user interface

    Learn about the Salesforce user interface, including the navigation, layout, and customization options.

  • 04. Learn about the Salesforce automation

    Learn about the Salesforce automation features, including workflow rules, validation rules, and process builder.

  • 05. Learn about the Salesforce security model

    Learn about the Salesforce security model, including profiles, permission sets, and sharing rules.

  • 06. Learn about the Salesforce analytics features

    Learn about the Salesforce analytics features, including reports and dashboards.

  • 07. Learn about the Salesforce integration

    Learn about the Salesforce integration options, including the Salesforce API and web services.

  • 08. Learn about the Salesforce deployment

    Learn about the Salesforce deployment options, including the Salesforce Sandbox and the Salesforce AppExchange.

  • 09. Learn about the Salesforce customization

    Learn about the Salesforce customization options, including Apex and Visualforce.

  • 10. Practice building real-world projects

    Practice building real-world projects and experimenting with different features of Salesforce, while also getting hands-on experience with troubleshooting and resolving issues.
